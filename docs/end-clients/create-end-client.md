# Create an end client

Go to **GodMode → End clients → + New end client**, or via **Resellers → [reseller] → End clients → + New**.

## Required fields

- **Full name**
- **Email address** — used for login and notifications
- **Reseller** — the reseller account this client belongs to

## Optional fields

- Company name
- Credit limit override (caps how many credits this client can consume)

Once created, an invitation email is sent automatically.
