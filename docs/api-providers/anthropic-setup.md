# Anthropic setup

Anthropic's Claude models are available as an alternative or primary LLM provider for brand naming, copywriting, and brand voice generation.

## Getting an API key

1. Log in to [console.anthropic.com](https://console.anthropic.com).
2. Go to **API Keys → Create Key**.
3. Name the key and copy it.

## Configuring in GodMode

1. Go to **GodMode → API providers → + Add provider**.
2. Select **Anthropic** as the provider type.
3. Paste your API key.
4. Select **Large language model** as the service category.
5. Save.

## Recommended model

Use `claude-sonnet-4-6` for production workloads — it offers the best balance of quality, speed, and cost for brand generation tasks.
