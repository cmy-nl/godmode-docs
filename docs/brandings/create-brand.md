# Create a brand

1. Go to **GodMode → Brandings → + New brand**.
2. Enter a **brand name** (internal reference, not shown to end clients).
3. Upload the **logo** (SVG or PNG, recommended minimum 400×120px).
4. Upload the **favicon** (ICO or PNG, 32×32px).
5. Set the **primary** and **secondary colors** using the color picker or by entering hex values.
6. Choose a **font** from the Google Fonts list or upload a custom font file.
7. Set the **custom domain** if applicable. See [Custom domain setup →](/brandings/custom-domain).
8. Configure the **email sender** name and address.
9. Click **Save brand**.

## Assigning a brand to an instance

After creating a brand, you need to assign it to one or more instances:

1. Go to **GodMode → Instances** and open the target instance.
2. In the **Branding** field, select the brand profile.
3. Click **Save**. The instance picks up the new branding within 60 seconds (no restart required).
