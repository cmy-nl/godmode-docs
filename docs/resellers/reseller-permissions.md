# Reseller permissions

| Action | Reseller admin |
|---|---|
| Create and manage end clients | ✅ |
| Configure their own branding | ✅ |
| Top up credits (self-service) | ✅ |
| View their own credit usage | ✅ |
| Enable / disable services for end clients | ✅ |
| Access GodMode Super Admin | ❌ |
| View other resellers | ❌ |
| Change API provider settings | ❌ |
| Issue or manage licenses | ❌ |
