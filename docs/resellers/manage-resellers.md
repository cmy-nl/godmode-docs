# Manage resellers

## Reseller list view

The Resellers list shows all reseller accounts with their status, assigned instance, license expiry, and credit balance. Filter by status, distributor, or country using the controls above the list.

## Reseller detail view

Click a reseller to open their detail view, which contains:

- **Overview** — status, instance URL, admin contact, package
- **Credits** — current balance, consumption history, top-up controls
- **License** — current license key, status, expiry date
- **Branding** — link to their branding configuration
- **End clients** — list of end clients under this reseller
- **Audit log** — actions taken by or on behalf of this reseller

## Common actions

### Topping up credits
Go to the reseller detail view → Credits tab → **Top up**. Enter the amount and confirm. Credits are added immediately.

### Suspending a reseller
Click **Suspend** in the reseller detail view. The reseller admin is locked out and their end clients lose access to the BrandForge platform. The instance remains provisioned and can be reactivated.

### Changing the package
Go to the reseller detail view → **Edit** → change the Package field. Credit tier adjustments take effect immediately.
