# Credit tiers

Credits are the unit of consumption on BrandForge. Every AI-powered action consumes a defined number of credits:

| Action | Credits consumed |
|---|---|
| AI Brand Setup (one-time) | 50 |
| Logo generation | 10 |
| Web-to-print document | 5 |
| Social media kit generation | 15 |
| Site builder AI page | 20 |

These rates are set at the platform level in [Settings → Services](/settings/general) and apply across all packages.
