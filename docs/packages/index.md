# Packages

Packages define the commercial offering assigned to each reseller — which credit tier they are on, which services are included, and what volume discount structure applies. Every reseller must have one package assigned.

:::tip Where is this in the UI?
Navigate to **GodMode → Packages** in the left sidebar.
:::

## BrandForge standard packages

BrandForge ships with three default packages, following the Starter / Professional / Business model:

| Package | Credits included | Target reseller type |
|---|---|---|
| **Starter** | 500 credits/mo | Small agencies, single-brand resellers |
| **Professional** | 2,000 credits/mo | Mid-size hosting providers |
| **Business** | 10,000 credits/mo | Large registrars, telcos, distributors |

Custom packages can be created for enterprise partners with specific volume requirements.

## What a package controls

- **Credit allocation** — monthly credits included in the package
- **Top-up pricing** — the per-credit cost for credits purchased above the included amount
- **Service access** — which BrandForge services (site builder, web-to-print, social kit) are enabled
- **Volume discount tier** — the discount bracket applied to bulk credit purchases (see [Volume discounts →](/packages/volume-discounts))
- **White-label options** — whether the reseller can use a custom domain, custom email sender, etc.

## Pricing commitment

BrandForge operates a **forever pricing** commitment: once a reseller is on a package, their per-credit price does not increase for the lifetime of their account. New packages may be introduced for new customers, but existing package holders are never repriced.
