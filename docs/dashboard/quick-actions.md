# Quick actions

Quick actions appear as a button row on the dashboard and give you one-click access to the most common GodMode tasks.

## Available quick actions

- **New reseller** — opens the create reseller form. See [Create reseller →](/resellers/create-reseller)
- **Issue license** — opens the license issuance flow. See [Issue a license →](/licenses/issue-license)
- **Top up credits** — opens the credit top-up panel for any reseller
- **New instance** — starts the instance provisioning wizard. See [Create instance →](/instances/create-instance)
