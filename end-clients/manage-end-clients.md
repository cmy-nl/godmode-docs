# Manage end clients

The end client list can be filtered by reseller, status, or creation date.

From the end client detail view you can:
- Edit contact details
- View credit consumption history
- Suspend or reactivate the account
- Impersonate the client (Super Admin only) to debug issues

:::caution Impersonation
Impersonation is logged in the [Audit log →](/docs/audit-logs) and should only be used for support purposes.
:::
