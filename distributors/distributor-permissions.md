# Distributor permissions

Distributor accounts have a fixed, scoped permission set. The table below shows what a distributor admin can and cannot do.

| Action | Distributor admin |
|---|---|
| Create resellers (within seat limit) | ✅ |
| Allocate credits to resellers | ✅ |
| View reseller usage and billing | ✅ |
| Suspend a reseller | ✅ |
| Access Super Admin panel | ❌ |
| Change platform API providers | ❌ |
| View other distributors | ❌ |
| Issue licenses directly | ❌ |

Permissions cannot be customized per distributor in the current version. Role customization is on the roadmap.
