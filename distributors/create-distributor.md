# Create a distributor

## Steps

1. Go to **GodMode → Distributors** and click **+ New distributor**.
2. Fill in the required fields:
   - **Company name** — the distributor's legal or trade name
   - **Admin email** — the primary login for the distributor admin account
   - **Country** — used for billing and VAT purposes
3. Set the **credit allocation** — the number of credits to assign from the Super Admin pool.
4. Set the **reseller seat limit** — the maximum number of reseller accounts this distributor can create.
5. Click **Create distributor**. An invitation email is sent to the admin email address.

## After creation

The distributor admin receives an email with a link to set their password. Once they log in, they see their scoped Distributor panel — not the full GodMode interface.

:::note
You can edit the credit allocation and reseller seat limit at any time from the distributor detail page.
:::
