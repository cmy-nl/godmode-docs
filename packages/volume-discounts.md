# Volume discounts

BrandForge uses an OpenSRS/Enom-style volume discount model: resellers that purchase larger credit top-ups receive a lower per-credit price.

## Default discount tiers

| Top-up amount | Discount |
|---|---|
| < 1,000 credits | Standard price |
| 1,000 – 4,999 credits | 10% off |
| 5,000 – 19,999 credits | 20% off |
| 20,000+ credits | 30% off |

These tiers can be customized per package or per reseller from the package detail view.
