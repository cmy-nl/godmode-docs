# Create a package

1. Go to **GodMode → Packages → + New package**.
2. Set a **name** and optional description.
3. Configure the **credit allocation** (monthly included credits).
4. Set the **top-up price per credit** for this package tier.
5. Select which **services** are enabled for resellers on this package.
6. Set the **volume discount tier** if applicable.
7. Click **Save package**.

The package is immediately available to assign to new or existing resellers.
