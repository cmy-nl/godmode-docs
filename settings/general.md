# General settings

## Platform details

| Setting | Description |
|---|---|
| **Platform name** | The internal name for this GodMode installation |
| **Support email** | Contact email shown to resellers for platform support |
| **Default language** | Default language for the platform UI and emails |
| **Timezone** | Server timezone used for timestamps and scheduled jobs |

## Credit cost configuration

Set the default credit cost per action for each service. These values are the platform defaults and can be overridden per service in **GodMode → Services → [service] → Configure**.

## Grace period

Set how long (in hours) an instance remains accessible after its license expires or is suspended. Default is **48 hours**.

## License expiry reminders

Configure how many days before license expiry reminder emails are sent to resellers. Default is 30 days and 7 days.
