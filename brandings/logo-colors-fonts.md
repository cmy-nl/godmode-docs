# Logo, colors and fonts

## Logo specifications

| Format | Recommendation |
|---|---|
| SVG | Preferred — scales perfectly at any size |
| PNG | Minimum 400×120px, transparent background |
| Max file size | 2 MB |

The logo appears in the top navigation bar of the reseller portal and in email headers. A separate **logo mark** (icon-only version) can be uploaded for use in compact layouts and the favicon slot.

## Colors

Two colors are required:

- **Primary color** — used for primary buttons, active navigation items, links, and progress indicators.
- **Secondary color** — used for secondary buttons, hover states, and decorative accents.

Colors are applied via CSS variables throughout the platform. Both dark and light mode variants are generated automatically from the primary and secondary values.

## Fonts

Select a font from the embedded Google Fonts browser, or upload a custom font file (WOFF2 format). The selected font is applied to headings and body text across the reseller portal.

If no font is set, the platform defaults to the system sans-serif stack.
