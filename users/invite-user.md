# Invite a user

1. Go to **GodMode → Users → + Invite user**.
2. Enter the user's **email address**.
3. Select their **role**. See [Roles and permissions →](/docs/users/roles-permissions) for a full breakdown of what each role can do.
4. If the role is **Distributor admin**, select which distributor account to associate them with.
5. If the role is **Reseller admin**, select which reseller account to associate them with.
6. Click **Send invitation**.

The user receives an email with a link to set their password. Invitation links expire after **72 hours**. Resend from the user list if the link expires.

## Multiple admins per reseller

A reseller account can have more than one reseller admin. Invite additional admins by selecting the same reseller in step 5. All admins for a reseller have equal access within the reseller dashboard.
