# Dashboard

The GodMode dashboard is your real-time overview of the entire BrandForge platform. It surfaces the metrics that matter most — active instances, license health, credit consumption, and recent activity — so you can spot issues and take action without digging through sub-sections.

:::tip Where is this in the UI?
Navigate to **GodMode → Dashboard** from the left sidebar. It is the default landing page after login.
:::

## What the dashboard shows

### Platform health
A top-level summary of the overall system status:
- Total active instances and their health indicators
- Total active licenses (valid / suspended / expired)
- Credit pool consumption across all resellers
- Any instances with unresolved provisioning errors

### Key metrics cards

| Metric | Description |
|---|---|
| **Active instances** | Instances in a running state, linked to a valid license |
| **Active licenses** | Licenses that are currently valid and not suspended |
| **Total resellers** | Reseller accounts across all distributors |
| **Credits consumed (30d)** | Platform-wide credit burn over the last 30 days |

### Activity feed
A chronological log of the most recent actions taken across the platform — new reseller signups, license issuances, credit top-ups, and branding changes. See [Activity feed →](/docs/dashboard/activity-feed) for details.

### Quick actions
Shortcuts to the most common GodMode tasks, available directly from the dashboard without navigating into sub-sections. See [Quick actions →](/docs/dashboard/quick-actions).
