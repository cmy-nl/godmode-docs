# Reseller dashboard overview

When a reseller admin logs in, they land on their own scoped dashboard — not the GodMode Super Admin panel. This page describes what the reseller dashboard looks like from the Super Admin's perspective, so you can support reseller partners effectively.

## What resellers see

- **Credit balance** — current credits remaining and a consumption chart
- **End clients** — list of their customers with access status
- **Branding** — their brand configuration (logo, colors, domain)
- **Services** — which BrandForge services are enabled for their instance
- **Support** — a contact form that routes to you as the platform operator

## What resellers cannot see

- Other resellers or distributors
- Platform-level API provider configuration
- License management (they see their own license status only)
- GodMode Super Admin sections
